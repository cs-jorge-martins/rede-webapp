Token de Identificação para API Kaplen

O Token de Identificação é um sequencial necessário para acessar os serviços disponíveis pela Kaplen via API.
Para garantir a autenticidade do cliente, a API de serviços exige que seja informado um Token de identificação, no Header da requisição Http que o sistema do cliente está montando e que será repassada para a URL dos serviços.
Para maiores informações, consultar a Documentação de Integração da Kaplen.
